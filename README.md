# Meu Primeiro Projeto Grande (API Java Spring)

Este projeto foi desenvolvido em Java utilizando o ecossistema Spring Boot. Foi um marco na minha jornada, realizado durante um evento intensivo de Java Spring em 2025.

Adiciono este README como forma de documentar minha evolução e orientar quem acessa este repositório, garantindo que não se percam **vendo** apenas arquivos soltos.

## Tecnologias Utilizadas

1.  **Linguagem Java:** Minha primeira interação robusta com a linguagem, desmistificando sua complexidade através da prática.
2.  **Spring Boot / Spring Tools:** Utilizado para acelerar o desenvolvimento com sua estrutura de injeção de dependências e configuração automática.
3.  **API REST:** Construção de servidor local e endpoints para comunicação.
4.  **PostgreSQL:** Banco de dados relacional conectado à aplicação para persistência das tabelas.
5.  **Postman:** Ferramenta essencial utilizada para testar as requisições (CRUD) e validar os retornos da API.
6.  **Docker:** Uso do Docker Compose para "containerizar" o banco de dados, facilitando o ambiente de desenvolvimento.

## Aprendizado

Este projeto representou meu *start* no mundo corporativo da programação. Tive meu primeiro contato prático com conceitos fundamentais como **CRUD**, **Conexão com Banco de Dados**, **Containerização com Docker** e testes de API.
